# **App Name**: Adigon AI Assistant

## Core Features:

- AI Interaction: Gemini API Integration: Allows users to interact with the Gemini AI model for generating creative content, answering questions, and providing assistance.
- AI Customization: Personalized Settings: Allow users to customize the AI's persona and behavior through options like tone selection, gender preference, relationship definition, and behavioral descriptions, influencing the AI's responses.
- Developer Mode: Code Generation Interface: When enabled, a toggle in settings unlocks a dedicated code generation interface, enabling users to request advanced code snippets or functions in various programming languages.
- Dynamic Prompts: Prompt Engineering Tool: Intelligently constructs prompts based on user input, conversation history, and AI Partner's persona, leveraging past context to obtain better output from the LLM. This is a tool.
- Conversational UI: Chat Interface: Displays conversation history and AI responses in a clear and intuitive chat-like interface, enhancing user interaction and readability.
- User Authentication: User Authentication: Provides email/password sign-up/login and Google Sign-In for secure user management and access to personalized settings.

## Style Guidelines:

- Background color: Deep charcoal (#121212) creates a sophisticated dark theme.
- Primary color: Muted blue (#64B5F6) is used for interactive elements and highlights, adding a touch of vibrancy without being overpowering.
- Accent color: A subtle green (#81C784) is employed for specific CTAs and status indicators, offering visual contrast and feedback.
- Font: 'Inter' (sans-serif) is used for both body text and headlines, providing a clean, modern, and readable experience.
- Use sleek, modern icons that represent AI and intelligence concepts. Ensure icons are simple, easily recognizable, and consistent in style.
- Rounded Corners: Apply rounded corners to buttons, cards, and message bubbles for a soft and contemporary feel.
- Use subtle animations for screen transitions, AI response generation, and button presses to enhance the user experience without being distracting. Must have advance super smooth animated effects also
